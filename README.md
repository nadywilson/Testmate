# TestMate 🚗

**Namibia Learner's Licence Practice Platform**

A full-stack web application to help Namibians prepare for their learner's licence test.

## Features

- 📚 Study materials by topic with images and PDFs
- ✅ Topic quizzes with instant feedback
- ⏱️ Full timed practice test (50 questions, 60 minutes)
- 🤖 AI-powered chatbot for road rule questions
- 📊 Progress tracking and weak area identification
- 🔐 Admin panel to manage questions and materials
- 👥 Separate login for Administrators and Learners

## Topics Covered

- Road Signs
- Traffic Rules
- Speed Limits
- Parking Rules
- Vehicle Controls

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL (via XAMPP)
- **AI:** Anthropic Claude API

## Local Setup

1. Install XAMPP
2. Clone this repo into `C:/xampp/htdocs/testmate`
3. Open phpMyAdmin and create a database called `testmate_db`
4. Import the SQL from the database export
5. Start Apache and MySQL in XAMPP
6. Go to `http://localhost/testmate`
